Application Infrastructure

This website build on Java 8 Spring Boot & Angular 13. Using Angular Material Design, https://material.angular.io/ and production-grade Spring based Application https://spring.io/projects/spring-boot. This is a freelance project that I have developed to better understand Java's most popular Spring Boot as a RESTFUL API and my potential delivery capabilities.

Java Spring Boot

a Restful API build with

Spring Security generating JWT token with authentication, validity and expiry
JPA Hibernate to create Entities that communicate to data-source via Repositories.
JDBC Java Database Connectivity (JDBC), an interface to exchange data with MS SQL.
MS SQL a relational database management system to store data.
JUnit 5 Testing developer-side testing on the JVM as the API is being build.

Task List

a todo-table that keeps track of immediate tasks, deliveries and deadlines. A useful tool to better manage weekly expectations set by team.

Task gets highlighted when due-date is less than a week away.

Enhancements could include a notification or email sent when task is due.

Built using Mat-Table from Angular Material Design

Kanban

Customized kanban to keep project deliveries and task status up-to-date.

Drag & drop approach to task management. Team mates update their task status as they work towards delivery. Every progress recorded with time-stamp.

Teams could use this kanban for their daily Sprint meetings. Project Lead would have a Bird's Eye view on road-blocks any team mate is facing.

Enhancements could include a statistical chart on team's average delivery time, drilling down to individual average completion timeline.

Built using CDK Drag-Drop from Angular Material Design
